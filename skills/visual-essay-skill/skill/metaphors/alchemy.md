# Metaphor Engine: ALCHEMY / FLUID

**Concept**: Mapping transformation, chemistry, and intangible states.
**Use When**: Explaining change, synthesis, emotion, or abstract "magic."

## Mappings
*   **Transformation** -> `Transmutation / Lead to Gold / State Change`
*   **Mixture** -> `Solution / Suspension / Emulsion`
*   **Purification** -> `Distillation / Filter / Fire`
*   **Volatile Idea** -> `Gas / Smoke / Explosion`
*   **Solid Fact** -> `Crystal / Stone / Precipitate`

## Visual Vocabulary
`glass beakers`, `colored smoke`, `liquids mixing`, `crystals`, `fire`, `steam`, `gold dust`, `mercury`.
