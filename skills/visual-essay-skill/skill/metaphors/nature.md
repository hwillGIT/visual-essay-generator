# Metaphor Engine: NATURE / ORGANIC

**Concept**: Mapping complex systems to biological or ecological processes.
**Use When**: Explaining growth, adaptation, networks, decay, or ecosystems.

## Mappings
*   **Structure/Organization** -> `Tree / Root System / Mycelium`
*   **Data/Flow** -> `River / Circulatory System / Rain`
*   **Conflict/Competition** -> `Predator-Prey / Overgrowth / Storm`
*   **Idea/Innovation** -> `Seed / Bloom / Spark`
*   **Legacy/History** -> `Fossils / Strata / Rings of a Tree`

## Visual Vocabulary
`roots`, `vines`, `blooming flowers`, `decaying logs`, `flowing water`, `storms`, `sunlight breaking through clouds`, `cellular structures`.
