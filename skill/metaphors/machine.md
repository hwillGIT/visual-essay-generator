# Metaphor Engine: MACHINE / MECHANISM

**Concept**: Mapping processes to gears, circuits, and engines.
**Use When**: Explaining logic, cause-and-effect, algorithms, or industrial systems.

## Mappings
*   **Logic/Reasoning** -> `Gears / Clockwork / Circuit Board`
*   **Energy/Motivation** -> `Battery / Fuel / Steam`
*   **Friction/Problem** -> `Rust / Broken Cog / Short Circuit`
*   **Output/Result** -> `Product / Light / Movement`
*   **Complexity** -> `Tangled Wires / Infinite Gears`

## Visual Vocabulary
`brass gears`, `circuit traces`, `steam engines`, `clock faces`, `levers`, `conveyor belts`, `robots`, `wires`.
