# Metaphor Engine: ARCHITECTURE / STRUCTURE

**Concept**: Mapping ideas to physical spaces, buildings, and construction.
**Use When**: Explaining foundational concepts, arguments, stability, or "building" something.

## Mappings
*   **Core Thesis** -> `Foundation / Cornerstone / Bedrock`
*   **Arguments** -> `Pillars / Walls / Buttresses`
*   **Connection/Transition** -> `Bridge / Staircase / Doorway`
*   **Perspective** -> `Window / Balcony / Tower`
*   **The Future/Unknown** -> `The Horizon / Unfinished Wing / Blueprint`

## Visual Vocabulary
`stone pillars`, `blueprints`, `scaffolding`, `arches`, `windows`, `staircases`, `doors ajar`, `ruins`, `skyscrapers`.
